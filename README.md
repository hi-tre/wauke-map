# Wauke · Nā Wahi Ulu
### Living Map of Hawaiʻi — Kapa Practitioner Resource

**[→ Open live map](https://hi-tre.github.io/wauke-map/)**

An interactive map tracing where wauke (*Broussonetia papyrifera*, paper mulberry) is cultivated across the Hawaiian Islands — a living tool for kapa practitioners, cultural educators, and restoration ecologists.

---

Wauke is the primary barkcloth plant of Hawaiʻi. This map documents active cultivation sites, potential growing areas, and ongoing study zones across five islands, organized by traditional moku districts.

**Islands covered:** Oʻahu · Maui · Hawaiʻi · Kauaʻi · Molokaʻi
